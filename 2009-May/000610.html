Hi Kari, Johan and all,

This morning I burned a clean installation CD for Xubuntu 
9.04 and then did the following:

1. Booted the CD into Xubuntu 9.04 in "live CD" mode.
2. Used the Synaptic package manager to install
	subversion
	python2.6-dev
	python-pmw
	portaudio19-dev
	libfftw3-dev
	libsamplerate0-dev
	gfortran
3. Installed g95 from the g95 web page
4. Downloaded the latest WSJT source code from the repository.

This all went very smoothly -- took about 20 minutes, max.

Then I compiled wsjt:

$ ./configure --enable-g95
$ make
$ python wsjt.py

WSJT runs perfectly.

Then

$ make clean
$ ./configure --enable-gfortran
$ make
$ python wsjt.py

WSJT starts but crashes within a few seconds.

It remains a mystery why Kari is able to compile the program 
with gfortran, and have it work, while I cannot.

BTW: I like Xubuntu.  Nice snappy user interface, no nonsense.
