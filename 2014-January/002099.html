> 
> The SVN repository presently has the sources for a version that,

> 
> 2. The lists of installed and runtime files, sections 10.1 and
> 10.2, need updating.
> 
> 3. All links need to be tested.
> 
> 4. All steps in the Tutorial need testing (preferable by someone
> other than me).
> 
> 5. Text written by Murray on "DXCC and Worked B4" needs to be
> integrated somewhere.
> 
> And finally, several new sections should probably be written:
> 
> 1. Cooperative Programs: basic description of the way WSJT-X works 
> together with JT-Alertm, Commander, and HRD.
> 
> 2. Frequenctly Asked Questions
> 
> 3. Troubleshooting
> 
> I'm very open to critiques of the User's Guide as it now stands,
> and to all suggestions on things that would improve it.
