Hi all,

I keep making the proverbial "one more change", but this needs to stop 
somewhere.  I've bumped the version number to 1.3, and I'm now proposing 
that we release v1.3 r3673 as the next "full release" of WSJT-X.

A windows package has posted at
http://physics.princeton.edu/pulsar/K1JT/WSJTX_130r3673.exe
for your last-minute perusal.  If you're using Linux, compile it yourself.

The current User's Guide is at
http://www.physics.princeton.edu/pulsar/K1JT/wsjtx-doc/wsjtx-main-toc2.html

If I hear no objections, tomorrow I will post these links on the WSJT 
web site and make an open announcement of availability.

I suggest that at this point a wsjtx1.3 branch should be made in the SVN 
repository.  If the released code needs any minor bug fixes or tweaks to 
its documentation, these can be done in that branch.

Bill, G4WJS, will then be free to move his new code for Rig-control, 
device setup, etc., into the main wsjtx branch.  Further development 
(other than bug fixes) will take place there.

Does this make sense to everyone?

	-- Joe, K1JT
