Hi Josh and all,

Have you been running all this time with no kvasd executable???  If so, 
you are losing some 2 dB of sensitivity in JT65 mode.

Go to
http://www.physics.princeton.edu/pulsar/K1JT/wsjtx-doc/wsjtx-main-toc2.html

Click on the left-edge link to Section 3.2 (Linux installation) and then 
on the link for downloading kvasd.

If this pre-built version does not run in your Linux, it may be 
necessary for me to build one for you.  As described in other recent 
communications on the wsjt-devel list, when necessary I've had others 
set up a login account for me on their machine.  If gcc and gfortran (or 
g95) are present, I can compile kvasd on that host in a few minutes.

     -- 73, Joe, K1JT
