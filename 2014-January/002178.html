The CMake scripts download kvasd from the SVN repo in binary form. The 
CPack extensions to the CMake scripts that will eventually build full 
release packages for each platofrom would also like to take advantage of 
that.

I suggest a tree of directories in SVN with versions of kvasd[.exe] for 
each platform we support, something like:

                           [kvasd-binary]
                                 |
    
+-----------------+----------+--------+---------------------+------------------------+
    |                 |                   |     |                        |
   [i386-pc-win32]   [i386-linux-gnu]    [x86_64-linux-gnu]    
[x86_64-apple-darwin]    [...]
     kvasd.exe         kvasd               kvasd      kvasd

That way they can all be stored, with history, in a single place, with 
their correct file names.
