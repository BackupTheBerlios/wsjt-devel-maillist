> Just to fill in a couple of gaps.
>
> I found that the 32-bit kvasd wouldn't run on a vanilla Ubuntu 13.10
> 64-bit. A quick check with 'ldd kvasd' showed that the library
> libgfortran.so.3 in 32-bit form was missing.
>
> Using:
>
> sudo apt-get install libgfortran3:i386
>
> was all that was required to enable kvasd to run. Clearly 32-bit support
> libraries aren't present in the current 64-bit Ubuntu but installing the
> Fortran library pulls in all the other required dependencies.
>
> bill at ubuntu-x64-vbox:~/test-install/bin$ dpkg --get-selections | grep -e
> "386"
> gcc-4.8-base:i386 install
> libc6:i386 install
> libc6-i386 install
> libgcc1:i386 install
> libgfortran3:i386 install
> libquadmath0:i386 install
> bill at ubuntu-x64-vbox:~/test-install/bin$
