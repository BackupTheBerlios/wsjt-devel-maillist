The WSJT Development Group is pleased to announce availability of a new 
release of WSJT-X -- Version 1.3, built from code revision 3673.

A link to the windows installation package can be found at
http://physics.princeton.edu/pulsar/K1JT/wsjtx.html

Changes from beta release v1.2.1, r3590 include the following:

1. An all-new WSJT-X User's Guide.  For a preview, see
http://www.physics.princeton.edu/pulsar/K1JT/wsjtx-doc/wsjtx-main-toc2.html

2. Improved handling of compound callsigns (e.g., ZA/K1ABC, G0XYZ/P).

3. Improved algorithm for flattening the displayed receiver passband.

4. Some protection against "wrong band" posts to PSK Reporter following
    a band change.

5. Fixes of two bugs that could cause a "hung decode" or abnormal
    program termination.

We recommend that all WSJT-X users should upgrade to version 1.3.

When upgrading it's permissible to install the new version into your 
existing WSJT-X directory, say C:\wsjtx.  In that case, your existing 
program settings will be used.

You might prefer to make a new directory, say C:\wsjtx1.3, thereby 
keeping your previous installation intact.  You can then copy your setup 
file wsjtx.ini and log file wsjtx_log.adi into the new directory.

We hope you will enjoy using WSJT-X v1.3!  Installation packages for 
Linux and OS X should be available very soon.

The Development Team is already hard at work on further program 
enhancements, to appear in the next revision.

	-- 73, Joe, K1JT -- for the whole WSJT Development Team
