3. Installation

3.1. Windows
WSJT-X can be downloaded from the WSJT Home Page . Click on the WSJT-X link at the left margin and then on the appropriate download link.

Execute the downloaded file and follow its installation instructions.

Install WSJT-X into its own directory (the suggested default is C:\WSJTX) rather than the conventional C:\Program Files\WSJTX.
any technical reason to make this recommendation?

By default all files relating to WSJT-X will be stored in your chosen installation directory and its subdirectories. You can uninstall WSJT-X simply by removing the installation directory and its contents.
