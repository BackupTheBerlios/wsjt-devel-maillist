I finally got wsjt-x compiled and running on Fedora
Heisenbug (20) as well as Rawhide.  I had to do the following:

     0.  Add your user ID to dialout and audio in /etc/group; reboot
     1. Load a bunch of libs
     2. cd lib; make -f Makefile.linux
     3. qmake-qt5
     4. make
     5.  find Pal* -type f - print | cpio -pmdvl ../wsjtx_install

     6.  run wsjtx from wsjtx_install directory
     7.   Configure wsjtx, adjust audio et al.

I made one contact each from Heisenbug and Rawhide on a 64 bit system.

What is kvasd?  wsjtx complains about it but works anyway.
