The way I set it up, the wsjt6_r315 directory must be just 
below your login directory, i.e., ~/wsjt6_r315. 
Alternatively, as I said in my email message yesterday, "If 
you install it somewhere other than your default login
directory, you will need to edit the first line in the
startup script "wsjt".  You would need to make this first 
line read:

export LD_LIBRARY_PATH=~/Desktop/Projects/test/wsjt6_r315

	-- Joe
