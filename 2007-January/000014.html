There was a bug in the EZ-install version of WSJT posted 
yesterday.  File names created while monitoring came up 
wrong most of the time.  I believe the problem is fixed in 
r317.

Please download
http://pulsar.princeton.edu/~joe/K1JT/wsjt6_r315.tgz
and use the following three commands to install and run WSJT 
5.9.6 r317:

    % tar xzf wsjt6_r317.tgz
    % cd wsjt6_r317
    % ./wsjt

I would appreciate learning whether this version runs OK on 
your computer.  I think it is, finally, a self-contained 
package that has everything needed to run WSJT on Linux.  It 
might even work on FreeBSD (Diane?) or Mac OS-X (Nick?).  It 
is *not* the way to go for anyone who wants to play with the 
source code, but it may serve very well for other users.

	-- 73, Joe, K1JT
