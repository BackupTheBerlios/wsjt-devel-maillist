Happy New Year Joe and all,

I installed wsjt6 according instructions. Opened the sample file and
wsjt crashed with error /see bellow/. That was with default Audio device
0 DSP. When I changed to Audio device 1 file was decoded fine and no
crash.

According fonts. Bitstream vera fonts are not installed by default in
Mandriva any more and I think in all newer distribution versions. So the
interface does not look very fine. That was the reason I changed fonts
in RPM to Helvetica and Serief respectively. Lucida typewriter is fine.

73, Harry LZ1BB
