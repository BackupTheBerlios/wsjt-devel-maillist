Hello Joe ,

This is request for a new function in WSJT-X and WSPR :

Would it be possible to add an option in WSJT-X and WSPR allowing to 
generate the output TX tone as an I/Q stereo signal instead of the 
current mono signal , together with a volume slider ?

This would allow us to feed the output Tx-signal of WSJT-X/WSPR directly 
to a Softrock RXTX kit without having to route the signal first to an 
external  software package for an I/Q conversion.

Linrad for example has excellent receiver functionalities but has no TX  
function to convert an input microphone signal into an I/Q signal and 
therefore can not be used in a  TX configuration with WSJT-X/WSPR and a 
SoftrockRxTx kit.

With this new function it would be possible to use Linrad and other 
"RX-only" SDR softwares   with the  Softrock RXTX kits by letting WSJT-X 
and WSPR feed the Softrock RXTX directly with the TX I/Q signal.

What do you think ?
