Hi John, Bill, and all,

I have written and tested a high-quality downsampling routine to convert 
from 48000 Hz to 12000 Hz sample rate.  It includes a good low-pass 
filter -- see the measured response curve posted at
http://physics.princeton.edu/pulsar/K1JT/fil4.pdf
The horizontal scale is in Hz, the vertical scale in dB.  For 
comparison, I included a curve showing the measured response of a 
downsampler that simply adds four consecutive samples.

Subroutine fil4() is intended to be called from within the function
Detector::writeData(), just before the store() call at line 52 in 
Detector.cpp.  Code in fil4() does the necessary things so there are no 
unwanted "edge effects" at the boundaries between successive buffers.

John, could you give this a try with your code that uses 48000 Hz input 
sampling?

	-- Joe, K1JT
