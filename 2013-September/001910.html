Hi Pierre,

ON5GN wrote:

> Would it be possible to add an option in WSJT-X and WSPR allowing to
> generate the output TX tone as an I/Q stereo signal instead of the
> current mono signal , together with a volume slider ?

WSPR has had full I/Q functionality for both Rx and Tx since version 
2.1, released in late 2010.  For user documentation see
http://physics.princeton.edu/pulsar/K1JT/WSPR_2.1_Supplement.pdf

There is also a (less publicized) WSPR 3.0, with documentation at
http://physics.princeton.edu/pulsar/K1JT/WSPR_3.0_User.pdf
and installation file at
physics.princeton.edu/pulsar/K1JT/WSPR3.01_r2513.EXE

We have talked about putting I/Q capability into WSJT-X. but have not 
yet written any code for same.

	-- Joe, K1JT
