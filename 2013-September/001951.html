Hi Hannu,

The module Audio.pyd is created by compiling the source code you 
downloaded.

In a suitably configured MinGW installation in Windows, with all 
necessary compiler tools present, this can be done with the command 
"make -f Makefile.MinGW".

Some other details are contained on the WSJT "Program Development" web page,
http://physics.princeton.edu/pulsar/K1JT/devel.html

	-- 73, Joe, K1JT
