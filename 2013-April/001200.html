I have been compiling WSPR under Fedora for some time.
The latest Fedora version requires the "load Image..." line in wspr.py
to be prefixed with "from PIL".  The following installs libs
needed for Gnuradio and WSPR:
date
set -x
yum --nogpgcheck  -y --skip-broken install gnuplot gifsicle giflib-devel 
ImageMagick-devel xfburn telnet-server xfce4-terminal xterm

yum --nogpgcheck  -y --skip-broken install portaudio-devel 
libsamplerate-devel *python-imaging* hamlib-devel numpy-f2py 
gcc-gfortran svn git

yum --nogpgcheck  -y --skip-broken install cmake boost-devel 
cppunit-devel fftw3-devel gsl-devel nasm yasm tiger* rss-glx* sphinx  
python-sphinx

yum --nogpgcheck  -y --skip-broken install wxPython-devel python-cheetah 
PyOpenGL  *qt4* *qt-4* tbb  qwt-devel PyQwt PyQt-devel *swig*

yum --nogpgcheck  -y --skip-broken install wxPython-devel python-lxml  
alsa-lib-devel  texlive-dvips libXaw-devel ncurses-devel xscreensaver-extras

echo DONE
date
