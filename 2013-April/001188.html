Hi Edson,

Thanks for working on the initialization code for shared memory and 
PortAudio.  Your changes seem to be working fine here.

One thought we might keep in mind: in the present code the size of the 
shared memory segment is dictated by the long T/R sequences of the slow 
JT9 submodes, in particular JT9-30.  Almost nobody is using those modes, 
as far as I know.  We might think about revising the code so that this 
memory is allocated on an "as needed" basis.
