Pleas heed the warning in the commit message for r3246.

"Polling interval" to 0 if you want unidirectional CAT control: the 
program then issues commands to the radio, but does not read the dial 
frequency.  Otherwise the dial frequency is read at the specified interval.

be very welcome!
