Hello Guillaume,

Seems to me like device permission problem, you might have to write a
udev rule, could be something like;

SUBSYSTEM=="usb", SYSFS{idVendor}=="0000", MODE="0666"

replace "0000" with the usb vendor id from lsusb command.

in a file named 51-prolific-rules in /etc/udev/rules.d/ directory

Hope this helps,

73
de Mike, VE2ExB
