I have an application where prior to starting WSJT or WSJT9 I want to predefine a message so that on starting the 13 character message is automatically sent.

I tried editing WSJT.ini and editing the following fields


Template1 RASPBERRYPI
Template2 %T_%M_%R
Template3 R%R
Template4 RRR
Template5 1234567
Template6 CQ_%M

with a  view that the system could automatically send template 1 on start. What I found was that the program didn’t pick up the changes when starting.
