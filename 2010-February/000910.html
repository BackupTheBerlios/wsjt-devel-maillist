Sorry to say, as far as I know we don't have anyone who can spend the 
time required to help others compile WSJT, WSPR, MAP65, or MAP65-IQ on 
an arbitrary OS or distribution.  We have put a lot of effort into using 
platform-independent tools and making sure that the programs can be 
built with free and/or open-source tools.  We make click-to-install 
packages for Windows and a number of *nix distributions, and we have put 
together lists of pre-requisite packages for the most widely used Linux 
distributions.  We know that WSJT, WSPR, etc., can be built on nearly 
*any* up-to-date unix-like OS.  Some users who have put in the necessary 
work have written "HowTo" files that go into the repository for the 
benefit of others.

While blazing a new trail, you may be on your own for a while.
