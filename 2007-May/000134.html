Harry, Diane, and all --

For what it's worth, for me WSJT 5.9.7 r383 compiles without 
problems on my Debian Linux machine, using the portaudio-v19 
code included with WSJT.  I use the following commands:

./configure --enable-portaudio \
--with-portaudio-include-dir=portaudio-v19/include \
--with-portaudio-lib-dir=/home/joe/wsjt/trunk/portaudio-v19/lib/.libs
make clean
make

Surely we can figure out what is going wrong in your case.

	-- Joe, K1JT
