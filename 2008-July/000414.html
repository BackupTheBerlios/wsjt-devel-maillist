Ubuntu package is simple recompiled one from Debian. KVASD file is not
presented in both distributions. The reason is that KV decoder is not a
free software and Debian policy does not allow using it. Ubuntu guys
just use to recompile Hamish's deb binary package. Just copy/paste in
the working folder KVASD. This will fix the problem.
