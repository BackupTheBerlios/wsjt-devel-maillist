Yesterday several questions were asked about the KVASD 
program.  Four executable files exist, as follows:

KVASD.EXE -- distributed with all versions of WSJT packaged
              for Windows.  Compiled with CVF.

The following are all on the WSJT Home Page, for example at

http://physics.princeton.edu/pulsar/K1JT/KVASD_g95.EXE

KVASD_g95.EXE -- compiled for Windows with MinGW and g95.

kvasd_gfortran -- compiled for Linux with gfortran

kvasd_g95 -- compiled for Linux with g95

Functionally the programs are identical.  If you're 
compiling for yourself, you should select one that executes 
on your system -- try executing it with the "-v" 
command-line option.  The executable file must be copied 
into a location somewhere in your $PATH.

If anyone needs an executable of KVASD for another platform, 
-- or if none of the files works for you -- please let me know.
