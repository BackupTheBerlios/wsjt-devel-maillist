After much recent work (and testing) by VA3DB, G4KLA, W1BW, W6CQZ, 
DL3LST, and K1JT (I hope I didn't forget anyone!), the WSPR program is 
nearly ready for a new release.  Since I do not foresee a need for 
additional features in the near future, I propose that we call it 
Version 1.0.

Today I merged some changes that I had made last week with Bruce's 
recent changes.  If I have not introduced errors in this process, I 
think version 1.0 is about done (as of r944).  If you are able to 
compile and run the program yourself, please give it a thorough 
exercising on your system in the next few days.  When we release it, I 
would like to be able to say that it is known to run well on Windows, 
Linux, FreeBSD, and Macintosh.  (Any others??)

Of course, if you find any problems on your platform, let me know!
