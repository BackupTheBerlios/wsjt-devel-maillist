For anyone else using Mandriva 2008.1:

The default bash profile in Mandriva 2008.1 has $HOME/bin set as a path 
though there isn't a /bin subdirectory in a users home directory by 
default. It was just a matter of adding the /bin subdirectory to my home 
directory, moving KVASD, KVASD_gfortran, and KVASD_g95 into it, making 
sure that all 3 permissions were set as executable. I should be set for 
either of the two Fortran compilers on the system now I hope.
