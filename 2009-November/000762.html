OK, I did find where to make it work, it is in tx.F90 file,

in 2 places, just before the  lines;
...
if(nport.gt.0) ierr=ptt(nport,pttport,1,iptt)    I have added
...
...
nport=1
if(nport.gt.0) ierr=ptt(nport,pttport,1,iptt)
...

This line is for the case where ptt_unix.c is used, the nport variable 
is not used in the case of compiling for unix, so that is probably why....

Hope that helps, everything works like a charm on an Ubuntu 64 system, 
aparts from little quirks but easily worked around, I can do some stuff 
in the shack while <weak beaconing>!
