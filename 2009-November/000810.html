The Debian package contains .svn subdirectories and as you say it should
not be installed in the root directory because of how the files are
arranged inside the package.  But someone somewhere is bound to install
it without reading these instructions.  Would it not be safer (and
simpler) to distribute the Linux WSPR binary as a tarball?  At the
moment little is gained by the .deb format.  Although it specifies
dependencies, those will not be automatically installed by dpkg if
missing.
