Dear Colleagues,

Recent activity in the WSPR branch of our repository is 
leading to a WSPR 2.0 that should be ready for public 
release soon.

I have just uploaded WSPR2 r1660 to the WSJT web site; the 
Windows installation file is at

http://physics.princeton.edu/pulsar/K1JT/WSPR2_r1660.EXE

and a simple .deb file usable on Ubuntu 9.04 (and perhaps 
other Debian-based systems) is at

http://physics.princeton.edu/pulsar/K1JT/wspr_2.00_r1660_i386.deb

To install and execute on Ubuntu:

$ sudo dpkg --instdir=. -i wspr_2.00_r1660_i386.deb
$ cd WSPR
$ ./wspr

[Note: this .deb file is a placeholder version, to be 
replaced eventually by one to be prepared by VK3SB.  Any 
chance of someone making packages for FreeBSD or OS X?]

WSPR 2.0 has a number of new features, including an
online User's Guide at 
http://physics.princeton.edu/pulsar/K1JT/WSPR_2.0_User.pdf . 
(Some editing is still to be done, and I need to add a short 
piece about Linux, FreeBSD, and Macintosh availability... 
but it's mostly complete.)

As always, your critical comments will be much appreciated! 
... this time, on both the software and the manual.  Of 
course, comments based on compile-your-own versions of the 
program are very welcome, as well as from the packages I made.

If you have been watching the recent developments, you may 
also have some comments on the feature list that made it 
into this hopefully close-to-final package.  I will be happy 
to hear them!

I hope that a public release of WSPR 2.0 can be made fairly 
soon.

     -- 73, Joe, K1JT
