You haven't told us what revision of WSPR you have been testing.  Are 
you compiling code yourself, or running something compiled by someone else?

If your WSPR revision level is anything older than about r1600, you are 
almost certainly chasing a bug that no longer exists.

If compiling your own, you need to check out source code directly from 
the SVN repository.  Otherwise, you need to get an updated package for 
your system built by someone else.  If your system is FreeBSD, that 
"someone" is probably Diane.
