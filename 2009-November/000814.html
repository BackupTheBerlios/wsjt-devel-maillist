OK, I'll try again.  I fixed the bug caught by Dave, G0DJA. 
  (r1677 always set dBm=0 in Tx messages).

So, I've now posted Windows and Ubuntu packages 
corresponding to r1682.  The packages are available at

Windows:
http://physics.princeton.edu/pulsar/K1JT/WSPR2_r1682.EXE

Ubuntu/Debian:
http://physics.princeton.edu/pulsar/K1JT/wspr_2.00_r1682_i386.deb
To install and execute on Ubuntu:

    $ sudo dpkg --instdir=. -i wspr_2.00_r1682_i386.deb
    $ cd WSPR
    $ ./wspr


I am still editing the WSPR 2.0 User's Guide, but you can 
read the present version at
http://physics.princeton.edu/pulsar/K1JT/WSPR_2.0_User.pdf

Bruce, W1BW, is working on updating the WSPRnet.org site so
is can accommodate several new features in WSPR 2.0.

With some luck, we may be ready to make a public release of
WSPR 2.0 by early next week.  If you have comments, have
found bugs, or have a last minute suggestion, now is the
time to let them be known!
