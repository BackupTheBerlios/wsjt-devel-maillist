Today I made a .deb file that seems to work for installing 
WSJT v7.04 r1311 on *ubuntu 9.04 systems.  I have no 
previous experience making .deb files, so I may have done 
some things in a non-standard way.  It would be very useful 
to have some feedback from others as to whether it works.

My own successful tests were done by downloading the .deb 
file from 
http://physics.princeton.edu/pulsar/K1JT/wsjt_7.04r1311_i386.deb
and then executing the following command:

$ sudo dpkg --instdir=. -i wsjt_7.04r1311_i386.deb

You might see messages about missing packages.  If so, use 
the Synaptic Package Manager to install them, and then 
re-execute the above command.  On a clean installation of 
*ubuntu 9.04 you will probably need to pre-install these 
packages:

   python-numpy
   python-tk
   python-imaging-tk
   libsamplerate0

To start WSJT, enter the commends

$ cd WSJT_DEB
$ ./wsjt

I don't know whether this procedure (or a similar one) will 
work on other Debian-based Linux distributions.  I will 
appreciate any reports of installation results, successful 
or otherwise!

For interest: the file Audio.so included in 
wsjt_7.04r1311_i386.deb was compiled using Intel Fortran 
v11.0, which is freely available for non-commercial use on 
Linux.
