Dear Colleagues,

Probably you have noticed today's activity in the wsjt/trunk 
directory.  The committed changes are intended to implement 
proper thread usage by protecting all Fortran I/O with mutex 
locks.

With revision 1277 the program compiles and seems to run 
correctly in both Linux (compiled with g95) and Windows 
(comopiled with CVF).  I have not yet tried Windows/MinGW/g95.

It does not run correctly in Linux/gfortran, for reasons I 
(still) do not understand.

I would appreciate hearing from anyone else who tries to 
compile and run r1277.
