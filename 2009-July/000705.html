> I have only looked in the fortran.f90 code that ends up in the python

> For re-entrant code, should you really be using the "save" fortran
> statement?

This is a reasonable question, and a reasonable thing to be
looking into.  Problems could arise if "save" is used
inappropriately in a routine that can be called from more
than one thread.

Most of the Fortran code is called in only one thread.  The
few exceptions should be examined carefully, to be sure they
are thread-safe.
