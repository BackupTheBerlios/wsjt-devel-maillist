Hi 
This seems to be a generic problem with USB on linux.
Create a group called usbusers and add yourself to the group.

Next in /etc/udev/rules.d
create a file 11-usbuser.rules
and in it this line
ACTION=="add", BUS=="usb", GROUP:="usbuser", MODE:="0666"

HTH, sorry I missed it earlier
