Hi all,

WSJT-X v1.1 r3389 has been running well here under both Windows and 
Linux.  Soon I would like to merge its code (now in the svn branch 
wsjtx_k1jt_p1) back into the main wsjtx development branch and make an 
alpha release of version 1.1 so that others can try it.

Question: has anyone else compiled and used v1.1, r3389 ?  If you have 
been compiling WSJT-X on your own, please give it a try and let me know 
how it runs for you.  It makes for a very nice way to make QSOs in both 
JT9 and JT65 -- both modes are decoded, and the Tx mode switches with 
automatically as required.
