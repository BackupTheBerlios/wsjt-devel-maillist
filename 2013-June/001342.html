To those who compile WSJT-X from code in the SVN repository:

It may be useful to summarize changes made between r3323 and the latest 
code of today, r3346:

1. Controls for for PTT Port and CAT Port are now wide enough to
display unix-style names like "/dev/ttyUSB0", etc.

2. Fixed an occasional program crash caused by receiving a message in
the form "call1 call2" (no grid, report, etc.).

3. Add /dev/ttyS4 - /dev/ttyS7 to list of linux serial devices.

4. If audio devices have not already been selected and saved, start
with the system default devices.

5. Actual QSO frequency is logged, instead of dialFreq only.

6. Added Tx Power to ADIF log information, with option to retain the
information between QSOs.

7. Added option to retain contents of Comments field between QSOs, for
both ADIF and wsjtx logs.

8. Added optional antenna description field for PSK Reporter.
(Presently implemented only for Windows, however.)

Please note: some of these features need more testing than they have
already received!  Please report here if you find any problems.

	-- 73, Joe, K1JT
