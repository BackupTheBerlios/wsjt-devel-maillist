wsjtx 1.0.r3323-2  Ubuntu packages are now available at
https://launchpad.net/~jnogatch/+archive/wsjtx

This includes fixes:
1. View->WaterfallPalette no longer causes seg fault.
2. Sample .wav file is now present.
3. Configure menu device name fields are wider, to show whole name

I announced the availability of these packages to the wsjt Yahoo group.

If you already installed the -1 package, all that should be needed is:
sudo apt-get update
sudo apt-get dist-upgrade


Debian packages have also been updated, but are still in the queue,
and not available yet.

73, John AC6SL
