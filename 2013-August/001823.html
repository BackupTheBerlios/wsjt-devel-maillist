Hi John,

Many distributions, Ubuntu included, tend to split libraries in separate
packages and sometimes things get messy. QtMultimedia for quite some time
has suffered from packaging problems (files scattered on separate
directories, inconsistent builds, etc.). I tried to get away from the
issues by compiling Qt from sources and putting the binaries under one
directory tree per version. This has an additional benefit to allow
multiple Qt versions installed on my system concurrent;y without files
being scattered all over the place.
