> Hi All,
Some feedback on how to fix this issue.
>
> I've followed the instructions on the berliOS site to put my SSH 
> public key in my accounts authorized keys and that works as expected, 
> except after a few hours the authorized keys file disappears. It looks 
> like there is some cron job recreating the account over and over.
>
> Before I raise a support ticket with berliOS.de does anyone know how 
> to get this to stick?
It turns out that the instructions on the web server are a little 
ambiguous. You can use ssh-copy-id to inject your public key but that is 
only temporary. To get a permanent setting; the only way is to use the 
key update window at the bottom of the setup page. That takes a few 
hours to get updated so the ssh-copy-id route is just to fill the gap.
