Hi all,

I have posted a Windows installation package for WSJT-X Version 1.2, 
r3537 at
http://physics.princeton.edu/pulsar/K1JT/WSJTX_120r3537.exe

If it's convenient for you to install this and give it a try, please do 
so, and let me know if you find any installation or startup issues.

If all is well, and if everyone is agreeable, I will plan to make a more 
public announcement of this package as a beta release, in a few days.

Here's a tentative list of differences between r3520 and r3537.

1. Audio I/O now uses Qt's built-in multimedia library inatead of PortAudio.

2. Windows reports to PSKReporter now use our own code, rather than 
PSKReporter.dll.

3. Many new optional palettes for waterfall colors, selectable from a 
drop-down list on the Wide Graph window.

4. Fixed bug causing misalignment of Cumulative spectrum when start 
frequency is not 0.

5. Tx message #6 is not selected at program startup.

6. Various other code changes for uniformity across platforms, ease of 
maintenance, and good programming practice.  (These should be invisible 
to users.)

7. UPdated to User's Guide.

8. Program is now built with the latest Qt, version 5.1.

Let me know if you think of things that I have omitted in this list!

	-- Joe, K1JT
