R3537 compiles and works well here on Mint 14 (Ubuntu 12.10) using QT 5.1.0.

The stream names that appear in pulse are built from the PID so pulse 
cannot remember their levels.  I've arranged things so my rx and tx 
levels are both default 100% but I can imagine this might be a small 
annoyance to others.

I've also just committed r3538, a small revision to logbook to make adif 
lookups faster by using a hash table instead of searching a list.  
There's also a small fix for ADIF files that use the data type flag.
