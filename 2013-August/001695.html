Hi all,

As Edson suggested, I think we have a milestone of sorts.  I have 
committed Bill's code for audio I/O, along with changes that make 
reporting to the PSKreporter site always use Edson's built-in code 
rather than the Windows DLL.

I am now using Qt 5.1 on both Windows and Linux.  I think it will not be 
possible to build this code exactly as it is under Qt4, because of some 
changes in the QAudio API.  (The necessary changes for Qt4 may be rather 
minor.)

I have bumped the version number to 1.2, and I'm suggesting that we make 
a new release fairly soon.  If you see any show-stoppers or have other 
reservations, please make them known!
