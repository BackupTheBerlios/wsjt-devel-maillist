Hi John,

Can you try running pavucontrol on Ubuntu 12.10 to see if your sound card
is listed? How did you get Qt5 inslalled? If from sources, do you have
libpulse-dev installed?

For testing, I like to have my own Qt compiled from sources. This
simplifies testing since it enables me to have multiple versions of Qt
installed concurrently. I like to compile Qt using the
--prefix=/usr/local/qt-x.x.x. This way all Qt files are organized in their
own directory tree (as opposed to scattering files all over the place).
Running the appropriate /usr/local/qt-x.x.x/bin/qmake is enough to select
the Qt version to test.
