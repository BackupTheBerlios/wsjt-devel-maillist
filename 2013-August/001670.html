That sounds as if you didn't start with r2522 sources. Maybe you had 
local edits in your working copies?

A patch (failed or otherwise) can be reversed out by using the same 
command you applied it with an additional '-R' argument, for example:

patch -p0 -R -i <patch-file-path>

If you have work in progress and want to try the patch I would suggest 
checking out a fresh source directory from r3522 rather than applying 
the patch to your current working set.
