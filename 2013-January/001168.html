> I've been trying to compile and install wsjt from trunk (version 9.3

The following commands check out and build WSJT v9 r2589 on my Linux box:

$ svn co -r2589 svn://svn.berlios.de/wsjt/trunk
$ cd trunk
$ ./configure
$ make

You can then start the program by typing

$ python wsjt.py

No doubt you can install the program in a standard "pythonic" manner, 
but I don't bother to do this.
