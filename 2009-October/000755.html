Hi all,

You will have noticed a lot of activity in the repository 
recently.  I continue to make progress with WSJT8, and
WSPR 2.0 is approaching a state at which it could be released.

I have two questions for the group.  Someone here almost 
certainly knows more that I about these things:

1. I have implemented a "make deb" command that creates a 
*.deb package that installs easily on Ubuntu (and possible 
other Debian-based linux distributions).  The package I made 
yesterday is on the WSJT web site at 
http://physics.princeton.edu/pulsar/K1JT/wspr_2.00r1596_i386.deb

If you download this file you can install and execute WSPR 
on Ubuntu with the command

% sudo dpkg --instdir=. -i wspr_2.00r1596_i386.deb
% cd WSPR
% ./wspr

I'm told that if you use GUI methods -- download the file to 
the Ubuntu desktop and double-click on it -- WSPR is 
installed under the root directory, i.e., at /WSPR.  Does 
anybody know how to change the .deb file so that it will go 
into a better place such as $HOME ?  Is this the best place?
