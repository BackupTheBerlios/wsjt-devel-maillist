It is proved howto compile wsjt on FC5 and FC6 by F1FBK. gmtine_ error
was well discussed. Please read previous messages on this list. As far
as I can rumember I solved this problem by installing f2py and scopy
from the original source, not from distribution packages:

---------------------------------------------------------
F2PY  ( Fortran to Python )
Download: http://cens.ioc.ee/projects/f2py2e/2.x/F2PY-2-latest.tar.gz

untar, then as root:
python setup.py install

---------------------------------------------------------
Scipy_Distutils
Download:  http://cens.ioc.ee/projects/f2py2e/2.x/scipy_distutils-latest.tar.gz

untar, then as root:
python setup.py install

---------------------------------------------------------
