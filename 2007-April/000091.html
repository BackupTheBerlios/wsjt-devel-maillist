On Debian,

The shared libraries are /usr/lib/libportaudio.so.0.0.18 and
/usr/lib/libportaudio.so.2.0.0, so there is no confict.

The header files and libportaudio.so link conflict, so of the two
development packages (libportaudio-dev and portaudio19-dev), only one
can be installed at a time.

This is standard on Debian - concurrent development targetting 
multiple library versions is not required, but running applications
build for multiple versions is.
