Everyone I know who has set up ~/.asoundrc more or less as follows:

$ cat /home/joe/.asoundrc

pcm.radio {
         type hw
         card 0
         device 0
}
pcm_slave.radioslave {
         pcm radio
         rate 48000
}
pcm.radioconv {
         type rate
         slave radioslave
}

... has found that selecting device "radioconv" for both input and 
output works just fine.
