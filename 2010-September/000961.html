Hi Clay,

Since the weekend is over, you've probably already figured it out.  In 
case not:

I compile for Windows using MinGW, Msys, and g95.  You need the Python 
and Subversion packages listed at

   http://www.physics.princeton.edu/pulsar/K1JT/devel.html

... (and maybe some other things I'm not remembering at the moment?)
To make the WSJT9.EXE single-file executable, you also need PyInstaller.

Once you get things set up, it's as simple as typing "make" in a 
command-prompt window.  But starting from scratch, wsjt is probably 
easier to build under linux than Windows.

	-- 73, Joe, K1JT
