Hello Everyone,

Just my $0.02 but CAT-based PTT wont' work on Yaesu FT-950's in SSB mode 
as it will only listen to the front MIC port and not the rear "DATA" 
port.  If the FT950 is places into PACKET mode, the CAT command will 
listen to the rear DATA port but then there are then specific 
limitations that I can't remember at the moment.  The solution for me 
when using SSB mode is to use a dedicated serial port keying up the DATA 
port's PTT line.  Works fine there and is automatically included with 
devices like the US Interface Navigator, Signalinks, etc.
