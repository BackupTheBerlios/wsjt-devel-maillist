I'm surprised you need to switch RTS on permanently with a USB 
interface. That is normally only needed with an interface that uses the 
RTS line to get power, USB devices are usually powered from the USB bus.

I suspect your radio may be set up for hardware flow control on the 
RS-232 port, if that were the case then the correct setting would be 
"Hardware" in the handshaking box and possibly checking the DTR box.

If you can confirm that and test if that works for you I would 
appreciate it.
