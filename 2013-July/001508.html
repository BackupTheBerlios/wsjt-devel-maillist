Test of WSJT-X v1.10 r3482a on Windows 7.

Reports being logged to ALL.TXT from
each of 3 instances.  Advanced->Multiple
Instances checked in each.  All are
controlling HDSDR instances via CAT,
but symptom is the same if CAT is not
used and window frequencies are manually
entered.

Note that all three instances log exactly
the same reports at 0215 - PY2RU, KK4ISJ,
and AA4RP.  These were actually received
by Instance #1, but appear in the band
activity window and ALL.TXT files for
all Instances.  This can be confusing,
since no signal appears on the waterfall of
two of the Instances at the reported audio frequency.

The spots to PskReporter appear to be
correctly sorted by band.  It's only the WSJT-X displays
and logs that contain the duplicate/crosstalked data.
