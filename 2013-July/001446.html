Hi all

Except for some harmless diagnostic write statements which will soon be 
removed, I think revision 3452 is a good candidate for beta release of 
version 1.1 to a wider group of users.

In case you haven't been following closely, changes from r3420 include 
the following:

1. CAT control via DxLab Commander

2. Enable Windows COM ports COM10 and above

3. Various tweaks to clean up the GUI, better format for the yellow Tx 
message, etc.

4. Lock Tx=Rx is now a checkBox

5. De-activate the change-over frequency selector and decode over the 
whole displayed band when in JT9-only or JT65-only mode.

6. Don't allow Tx mode to be changed while transmitting.

7. Double-click on yellow Tx message sets frequency, but nothing else.

8. Fix Tool Tip on JT65 | JT9 spinner control.

I'm working on bringing the User's Guide up to what's needed for Version 
1.1.

Feedback is especially welcome at this time!

	-- Joe, K1JT
