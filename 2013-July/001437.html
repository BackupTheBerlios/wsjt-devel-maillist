The main thrusts of recent development in the wsjtx_k1jt_p1 branch (aka 
"version 1.1") have been

1. To implement JT65 in a such a way that JT9 and JT65 can be active 
concurrently.

2. To realize the potential of radios that can provide displayable 
bandwidths up to 5 kHz.

I think version 1.1 has matured to the point where it's nearly ready to 
be merged back into the main WSJT-X development branch.  Anybody see any 
problem with this, in the near future?

Here's what I'm still working on, at present:

1. Rig control via DxLab Commander.

2. A version 1.1 User's Guide.

3. Odds and ends, including a nicer-looking "Thermo" control for Rx 
noise level indication.
