Joe,

I have passed a recipe to those who have asked concerning Mac OSX compilation.   The recipe is a set of URLs to be used to download the various libraries that are needed, e.g. fftw3, portaudio, Qt, hamlib etc.   The basic kit that must be in place should be compilers for C and Fortran.   (I note that G4WJS is not a Fortran fan (!) but Fortran definitely has its uses in places where the C or C++ equivalent is a pain....I assume you are not in any rush to convert lib/*.f90 to C...)     This recipe could, in principle, be posted on the K1JT development page.

I use gcc v 4.2.1  and gfortran which is also based on gcc 4.2.1

If there is a problem, I found that  a number of my correspondents are not comfortable to work in a Unix environment (which is the underpinning of Mac OSX).   I have a downloadable Mac OSX package of r3496 which is being tested at the moment by someone with a Mac without any relevant libraries.   I am waiting for results.

Other than Windows and Unix-based systems, what other platforms are there that any might be using?
