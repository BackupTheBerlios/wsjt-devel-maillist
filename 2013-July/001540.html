Hi David,

WSJT-X already has an undocumented feature that does at least some of 
what you want.  Put a file named "fonts.txt" in the same directory as 
the wsjtx executable, and make the contents of the file a single line of 
the form

   helvetica,10,50

The first word is the name of a font family.  The numbers are (relative) 
font size and weight.  When no such file is present, the default is the 
same as

   helvetica,8,50

At present, I think this setting affects all labels but not the decoded 
text windows.  If it were desirable, it would probably not be a big deal 
to scale those windows as well.

Let us know whether this is good for you.

	-- Joe, K1JT
