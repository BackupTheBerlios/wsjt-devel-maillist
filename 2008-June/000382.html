Hi Hamish,

I don't think Harry's problem has anything to do with sound.  According 
to his error message, the code giving him trouble is a simple Python 
request to move the contents of the WSJT "To Radio" box into the global 
common area accessible to Fortran.  Maybe he has a new Pyhon?  As a 
workaround, one could try replacing the line

     Audio.gcom2.hiscall=(ToRadio.get()+'            ')[:12]

with something like

     try:
         Audio.gcom2.hiscall=(ToRadio.get()+'            ')[:12]
     except:
         pass

just to see what happens.  As a diagnostic, try

         print ToRadio.get()

, ... etc., in place of "pass".

	-- Joe, K1JT
