PS to my note about kvasd_g95:

1. Remember that file names are case sensitive.  The command invoking 
kvasd_g95 should be something like

	iret=system('KVASD_g95 -q > dev_null')

2. Since it was compiled one of my Linux machines, it's possible that 
when run it will want some library routine from my (older) Linux.  If 
that turns out to be true, I'll need to compile it on a later Linux.

Does anyone know of a way to minimize such problems with code that 
cannot be compiled locally, by users?
