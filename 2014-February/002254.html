Hi Chuck,

Sorry for the name confusion there.

The svn checkout was my blunder, apologies for that. There are two
methods for checking out source code, be it documentation or application:


1). ** For Dev's with commit access ( docs ):
- -change user-name to your Berlios user-name

svn co https://user-name@svn.berlios.de/svnroot/repos/wsjt/branches/doc


2). ** For non-commit checkout ( docs ):
- - Used for building apps or docs, browsing over, etc. You cannot
commit via this URL

svn co svn://svn.berlios.de/wsjt/branches/doc


If you intend to check-in edits using [ svn commit ], you need to use
method 1). above.
