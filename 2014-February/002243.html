Hello All,

If anyone has some spare time, I could use some help testing an Windows
batch file that Auto-Installs Cygwin + Required packages for working on
WSJT docs.

I've tested this on WinXP-32 & Vista 32/64, but I do not have access to
Win7/8.

Total Install Time: ~8.5 min (including download time) on slow internet

If interested, use the following to test:

Note(s):

a. I attached the zip file, but I'm unsure sure how the boards handle
attachments, so I'm also providing a shared link to my Ubuntu One account:

b. This is a local folder only install, no shortcuts or system level
integration will should happen. To remove, simply delete the wsjtx-doc
folder.

1). ** Download: http://ubuntuone.com/5TXl9swZpkJyBXjlbzsI8G
MD5SUM: 5f08b4759a2232ef1dbc2b0b7723b133
SHASUM: 8a268ceb1bb90e7bd4527ee95ab7092748975977

wsjt-doc.zip contains two files:
- the folder itself:      wsjt-doc
- the windows batch file: wsjt-doc.bat

2). ** Unzip the file wherever you like c:\ d:\ \Desktop \Documents
USB-Key, etc. If using USB Key, install time is a good bit slower.
For the example, I'll use c:\wsjt-doc as an example here.

3). ** Download one the following installers, saving it to the
c:\wsjt-doc folder:

- http://cygwin.com/setup-x86.exe
- http://cygwin.com/setup-x86_64.exe

4). ** Browse to the c:\wsjt-doc folder, Double-Click on
wsjt-doc-setup.bat or Right-Click >> Open
- On my slow internet, it was less than 9min to download and install.

5). ** If the installer finishes successfully, Double-Click on
c:\wsjt-doc\wsjt-doc.bat to open the Cygwin terminal, then check out the
WSJT documentation with:

svn co svn://svn.berlios.de/wsjt/branches/wsjtx

6). ** Once downloaded, build the docs with: ./build-doc.sh wsjtx

7). ** If all went according to plan, you should have a copy of the
latest WSJT-X User Guide in ..doc/wsjtx/ folder

8). ** With your browser, open ..doc/wsjtx/wsjtx-main-toc2.html to
ensure it was built correctly.

If you run into any trouble, please send me a note with the details so I
can try and make this work across most of the Windows versions.
