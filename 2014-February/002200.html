Hi Uli,

Many thanks for your report.  We are well aware that some CAT control 
issues remain in WSJT-X v1.3.  They can be traced to the hamlib library 
functions for certain rigs, and we are working to get them fixed.  The 
next release of WSJT-X will have some major changes in the rig-control code.

In the meantime, you will need to run with the polling interval set to 0.
	-- 73, Joe, K1JT
