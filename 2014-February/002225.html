Dear colleagues,

I hope I've done this correctly.  The doc files for WSJT-X have been 
moved to a new branch in the SVN repository, separate from all code.

When you next do "svn update" in your working directory for code, the 
.../wsjtx/doc directory will be deleted.  (If you have unversioned files 
in there, you will probably need to remove them manually.)

To retrieve the doc files from their new location, you'll need to do 
something similar to the following:

svn co https://k1jt@svn.berlios.de/svnroot/repos/wsjt/branches/doc

The files will then be found in ./doc/wsjtx.  In future we will have 
similar direcories ./doc/map65, ./dos/wsprx, ... etc.

	-- Joe, K1JT
