Hi all,

Some interest has been expressed in a bare-bones, command-line version 
of WSPR to be used for tests and specialized applications.  I have 
therefore updated the program that has been available since the very 
beginning of WSPR, bringing its decoder up to the present WSPR standard.

To avoid unnecessary confusion with the GUI-driven WSPR, I call the 
command-line version "wspr0".  Source code for wspr0 is mostly Fortran, 
with a few routines in C.  All source is now in the SVN respository for 
WSPR.  Most of the routines used by wspr0 are the same as those used by 
WSPR, so there is no need for separate duplication.  The makefiles now 
build the command-line program as well as the GUI-driven one. 
Compilation and execution of wspr0 seems to work properly under both 
Windows and Linux.

I have updated the instruction file for the command-line program and 
posted it at

http://www.physics.princeton.edu/pulsar/K1JT/WSPR0_Instructions.TXT

It is also in the SVN reporisory, of course.

Feedback is welcome.

	-- 73, Joe, K1JT
