A bare-bones command-line WSPR has existed since March 2008.

Instructions:
http://www.physics.princeton.edu/pulsar/K1JT/MEPT_Instructions.TXT

Windows executable:
http://www.physics.princeton.edu/pulsar/K1JT/WSPR.EXE

Source code:
http://www.physics.princeton.edu/pulsar/K1JT/wspr0.tgz


To compile and execute in *nix, unpack the tarball and do

$ ./configure --enable-gfortran       (or --enable-g95 )
$ make
$ wspr

You will need to have installed libportaudio2 and libportaudio-dev.


To compile and execute in Windows with MinGW:

$ cp Makefile.MinGW Makefile
$ make
$ wspr

Source code for the command-line version is essentially unchanged since 
March 2008.  Various improvements have been made to the decoder since 
then; these could certainly be imported into the command-line version. 
Differences in performance would probably be small, however.  I-Q mode 
and Frequency Hopping are not supported.
