Hi Edson,

When reading from a file, command-line wspr was looking for a file 
written by itself, with a full 120 s of data.

You want it to read from a filw written by GUI WSPR, which has only 114 
s of data.

On line 8 of file mept_rx.f90 you'll find the statement

   parameter (NMAX=120*12000)              !Max length of waveform

Change 120 to 114, and recompile.  All should be well.

	-- 73, Joe, K1JT
