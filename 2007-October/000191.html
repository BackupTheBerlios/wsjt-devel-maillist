I've found out now that python-imaging has nothing to do with python,
its python-ware, If you install Imaging it still gets a build fail from
Tinker, and libtix needs to be installed before Imaging is built.

to be depreciated.

to include the imaging libs, _imagingtk.so, as part of the package
