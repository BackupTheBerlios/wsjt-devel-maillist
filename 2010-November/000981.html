Hi all,

I have been running WSPR with Sivan's enhancement to allow frequency 
hopping.  It works very well here...

I've now added another feature: whenever WSPR executes a successful 
band-chance command to a CAT-controlled radio, it looks for a file named 
user_hardware.bat, user_hardware.exe, or user_hardware in the working 
directory.  If one of these is found, WSPR tries to execute the command

user_hardware nnn

where nnn is the band-designation wavelength in meters.

My station uses a USB-interfaced device called "LabJack U12" to control 
various bandswitching functions external to the radio.  In particular, 
it connects the radio to a selected antenna for each HF band.

The simple Python program "user_hardware.py" is an example of how my 
station can control antenna switching through the LabJack.  To enable 
this feature in a clean WSPR installation, I would need to add a batch 
file user_hardware.bat containing this single line:

python user_hardware.py %1

Other users may write their own station control program using any 
desired computer language.

I am now running WSPR with frequency hopping among 5 bands.

Comments welcome!
