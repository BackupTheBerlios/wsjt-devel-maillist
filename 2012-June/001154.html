From: Joe Taylor, K1JT

Many thanks to Les, W2LPL, and Carsten, DM1CG, for helping me to 
identify and correct a bug in recent revisions of MAP65 v2.3.0.

This bug appeared only when using direct soundcard inputto MAP65.  The 
symptom was that a weaker "ghost signal" appeared in the waterfall 
display in receive intervals following the reception of a very strong 
signal.

The bug has been fixed revision 2468, now available on the WSJT web site:

   http://physics.princeton.edu/pulsar/K1JT/map65.html

A direct link to the installation file is

   http://physics.princeton.edu/pulsar/K1JT/MAP65_230r2468.exe

As always, additional bug reports and feature requests for MAP65 will be 
very welcome!

     -- 73, Joe, K1JT
